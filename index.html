Drawing Tips

  • Click and drag to draw walls, doors, and windows
  • Double-click on a wall to add a door or window
  • Enable "Snap to Grid" for precise alignment
  • Use Ctrl+Click to select multiple elements
  • Right-click elements for quick options

Wall Types

  • Exterior Walls: Typically 6" thick, include insulation cavity
  • Interior Walls: Standard 4.5" thick walls with 2x4 framing
  • Partition Walls: Thin non-load bearing walls
